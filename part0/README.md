# Part 0 - Fundamentals of Web Apps

This part introduces the foundations of how web applications work.

## Key Tasks and Learnings

- Understand the client-server model and HTTP basics.
- Learn how browsers, servers, and APIs interact.
- Practice documenting app behavior with sequence diagrams.

## Exercises

- [Ex 0.4 - New Note Diagram](ex04-new-note-diagram.md)
- [Ex 0.5 - Single Page App Diagram](ex05-spa-diagram.md)
- [Ex 0.6 - SPA New Note Diagram](ex06-spa-new-note-diagram.md)

## Install Instructions

No installation is required for Part 0. The exercises are Markdown documentation and sequence diagrams.

## Usage Examples

Read the exercise files directly:

```bash
open ex04-new-note-diagram.md
open ex05-spa-diagram.md
open ex06-spa-new-note-diagram.md
```
