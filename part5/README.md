# Part 5 - Testing React Apps

This part focuses on frontend testing and advanced React application development.

## Key Tasks and Learnings

- Test React components and UI behavior.
- Test custom hooks and application logic.
- Handle authentication state and protected functionality in frontend apps.
- Improve confidence in UI changes through automated tests.

## Status

In progress

## Install Instructions

Install dependencies in each Part 5 project directory:

```bash
cd <part5-project>
npm install
```

## Usage Examples

Run a project in development mode:

```bash
cd <part5-project>
npm run dev
```

Run tests (when available in the project):

```bash
npm test
```
