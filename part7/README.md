# Part 7 - React Router, Custom Hooks, Styling, and Tooling

This part expands frontend architecture, navigation, and reusable abstractions.

## Key Tasks and Learnings

- Build multi-view React apps with client-side routing.
- Extract reusable logic with custom hooks.
- Improve UI structure and styling approaches.
- Work with configuration and tooling for larger React projects.

## Status

In progress

## Install Instructions

Install dependencies in each Part 7 project directory:

```bash
cd <part7-project>
npm install
```

## Usage Examples

Run a project in development mode:

```bash
cd <part7-project>
npm run dev
```

Run tests (when available in the project):

```bash
npm test
```
