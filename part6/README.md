# Part 6 - Advanced State Management

This part covers scalable state management patterns for React applications.

## Key Tasks and Learnings

- Use Redux for predictable application state.
- Work with asynchronous state updates and side effects.
- Compare and apply alternative patterns such as React Context and server-state approaches.
- Organize larger frontend codebases with clearer data flow.

## Status

In progress

## Install Instructions

Install dependencies in each Part 6 project directory:

```bash
cd <part6-project>
npm install
```

## Usage Examples

Run a project in development mode:

```bash
cd <part6-project>
npm run dev
```

Run tests (when available in the project):

```bash
npm test
```
