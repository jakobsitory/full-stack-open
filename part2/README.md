# Part 2 – Communicating with Server

React exercises covering rendering collections, forms, fetching data from APIs, and styling.

## Projects

| Project | Description |
|---------|-------------|
| [courseinfo](courseinfo/) | Refactored course info with separate components and `reduce` for totals |
| [phonebook](phonebook/) | Contact manager with CRUD operations via a JSON Server backend |
| [countries](countries/) | Country search app using the REST Countries API |

## Install Instructions

```bash
cd <project>
npm install
```

## Usage Examples

```bash
cd <project>
npm run dev
```
