# Part 1 – Introduction to React

Introductory React exercises covering components, props, state, and event handling.

## Projects

| Project | Description |
|---------|-------------|
| [courseinfo](courseinfo/) | Displays course information using nested components and props |
| [unicafe](unicafe/) | Feedback collector with statistics (good / neutral / bad) |
| [anecdotes](anecdotes/) | Random software anecdote viewer with voting |

## Install Instructions

```bash
cd <project>
npm install
```

## Usage Examples

```bash
cd <project>
npm run dev
```
