# Part 3 - Programming a Server with Node.js and Express

This part focuses on building and deploying a backend service and integrating it with a React frontend.

## Key Tasks and Learnings

- Build REST APIs using Node.js and Express.
- Use MongoDB and Mongoose for persistent storage.
- Add validation, error handling, and environment configuration.
- Connect frontend and backend into a full-stack application.

## Projects

- [phonebook-backend](phonebook-backend/)
- [phonebook-frontend](phonebook-frontend/)

## Install Instructions

Install dependencies separately for backend and frontend:

```bash
cd phonebook-backend
npm install

cd ../phonebook-frontend
npm install
```

## Usage Examples

Run backend (requires `PORT` and `MONGODB_URI` in environment):

```bash
cd phonebook-backend
npm run dev
```

Run frontend:

```bash
cd phonebook-frontend
npm run dev
```
